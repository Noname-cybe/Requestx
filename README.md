# Requestx
---

## Deskripsi :
- Hanya untuk tujuan pendidikan, gunakan lah dengan **bijak** !
- **Script ini hanya berjalan di Linux debian ( Kali Linux ).**

---

## Setup
```bash
git clone https://github.com/Noname-cybe/request2.git
cd Requestx
pip install -r requirements.txt
```
## Run
```bash
python d*os-v1.py
```
## Fix
**Kalo gak bisa, buat virtual environment dulu, sebelum install requirements :**
```bash
python3 -m venv venv
source venv/bin/activate
cd Requestx
pip install -r requirements.txt
python d*os-v1.py
```
